A hitch hiker guide to to becoming a programmer

Hi, if you are reading this either you are the curator or a student who is willing to learn programming. I know how it people get overwhelmed when they jump into the field of programming becuase there are a lot of terminologies and technologies popping up.

These are the things that you need to become good at programmig

  1. A working laptop or a PC with an internet connection
  2. Select a programming language
  3. Your time and attention

these are the only three things that you need to get started. Most of the time people get stuck at choosing a programming language.But i suggest you to start with Python, because it has a less steeper learning curve.But believe me, at the end you have to learn multiple languages. You cannot win in the long run. if you choose one language.technologies

Once you have selected a language. learn the basic syntax(loops, conditions,functions etc etc). Within a few weeks you will start getting comfortable with the syntax.Then start learning datastructures and algorithms as these two will get you the job.

I would highly recommend you to go for the project based learning. This will help you learn, how programming is used in real world.
